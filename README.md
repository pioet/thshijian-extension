# 清华研究生社会实践系统增强插件

## 功能

1. 新详情页跳转改进成内嵌侧边栏
2. 收藏页面增加查询已选人数情况
3. 已提交申请增加需求人数显示
4. 项目查询页面支持中签概率排序
5. 支持查看提交历史

## 更新记录

- 2026-05-12 本项目提交到 thuservices：[commit:fb34f0](https://github.com/thuservices/thuservices/commit/fb34f00ba7a176306a1c98f63ab3c26eff8f496a)。
- 2026-05-13 校方官方网站上线新功能「已提交申请增加详情页点击，已选人数情况」。为避免重复，本插件移除了该功能 [commit:d223b1](https://github.com/pioet/thshijian-extension/commit/d223b157179d3c2d395ab6d30e1b878105d97428)。校方后端调整了`需求人数`和`已申请人数`的接口格式，影响到本插件需求人数和已申请人数的功能，现已修复。

## 安装

1. Chrome 浏览器打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `extension` 目录
